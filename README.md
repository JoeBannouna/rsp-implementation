# RSP Implementation

## Structure
#### Linked Lists
- LinkedList.hpp
- LinkedList.cpp
- main-linkedlist.cpp

#### Binary Trees
- BinaryTree.hpp
- BinaryTree.cpp
- main-binarytree.cpp

#### Graphs
- Graph.hpp
- Graph.cpp
- main-graph.cpp

## How to run
```bash
make linkedlist
```
```bash
make binarytree
```
```bash
make graph
```
